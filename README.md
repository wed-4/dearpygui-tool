# Ragnarok
dearpyguiで開発された多機能なツール。課題研究に向けて開発中。言語は日本語にのみ対応しています。

Multifunctional tool developed using dearpygui. Under development for issue research. Language is Japanese only.

## 実行方法

    git clone https://github.com/wed-4/dearpygui-tool.git
    cd dearpygui-tool
    python main.py

以上のように実行してください。
gitを持っていない方は「Download ZIP」からzipファイルとしてダウンロードし、適当な場所に解凍してください。
